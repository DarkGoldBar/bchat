# bchat

这个项目是一个使用 Javascript 语言的 Vue 3 + Vuetify 3 + Vue-Router 的单页面项目。这是一个个人项目。这个项目主要面向手机端构建，同时兼顾电脑端的显示。使用 vue-cli 构建，添加了 PWA 支持。只面对现代浏览器，放弃了 Babel 支持。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
